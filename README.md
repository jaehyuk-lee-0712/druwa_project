# druwa_project

## client
````
cd /client/
npx create-react-app . 
npm i react-router-dom
npm i sass
npm i react-quill
npm install @emailjs/browser --save
npm i react-modal
````

## server
````
cd /server/
npm init -y 
npm i express (서버연결)
npm i mongoose (db연결)
npm i cors (cors에러)
npm i nodemon
npm i bcryptjs
npm i jsonwebtoken
npm i cookie-parser
npm i axios
npm i cheerio
````
